# matmarkpp

Markdown in c++ (Obsidian.md flavor)

## Implemented features

- [x] Headers
- [x] "[[]]" syntax for md
- [x] "![[]]" syntax for images
- [x] Horizontal lines
- [x] Unordered lists
- [ ] Ordered lists
- [ ] Standrad markdown links
- [ ] Tables


# Project structures

- `include`: contains files to be included
- `src`: source files

- `main`: files for running lib as standalone application
